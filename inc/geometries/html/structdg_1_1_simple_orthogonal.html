Detailed Description

template<class Psi, class PsiX, class PsiY, class LaplacePsi>
struct dg::SimpleOrthogonal< Psi, PsiX, PsiY, LaplacePsi >

Generate a simple orthogonal grid.

Psi is the radial coordinate and you can choose various discretizations of the first line

Template Parameters
PsiAll the template parameters must model a Binary-operator i.e. the bracket operator() must be callable with two arguments and return a double.
